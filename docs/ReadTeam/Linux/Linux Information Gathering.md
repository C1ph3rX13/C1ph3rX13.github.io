## Hidden

+ 取消环境变量 `HISTFILE` 的设置

```bash
unset HISTFILE
```

+ 将环境变量 `HISTFILE` 设置为 `/dev/null`

```bash
export HISTFILE=/dev/null
```

+ 彻底清除历史

```bash
history -c
```

+ 关闭Shell的历史记录功能

```bash
set +o history
```

- 打印本地端口开放信息

```bash

# -t 表示只显示TCP连接和监听端口
# -p 表示显示与每个套接字关联的进程ID（PID）和程序名称
```
